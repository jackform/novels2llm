- 一条记录的 text 中包含多个人的完整话语

## 输出格式
请只输出以下 JSON 格式，不要有其他内容：
```json
{{
  "dialogues": [
    {{
      "dialogue_index": 1,
      "speaker": "角色名",
      "listener": "角色名或null",
      "content": "对话内容",
      "context": "上下文",
      "is_inner_thought": false,
      "is_split": false,
      "original_index": null
    }}
  ]
}}
```

- `dialogue_index`：从1开始的序号（拆分的对话使用小数字号如 3a, 3b）
- `is_inner_thought: true` 的对话不会被纳入事件对话，但仍需输出（用于验证）
- `is_split: true` 表示该条是从原始候选拆分出来的
- `original_index`：拆分前的原始候选编号（非拆分条目为 null）
- 如果一条候选被判定为内心独白，仍需输出但标记 `is_inner_thought: true`
- 如果一条候选被拆分，每个拆分部分作为独立条目输出
